    Binding, why is it so interesting!

    The Binding function is very interesting because it allows autonomous operation between two devices, even if your home automation box fails. It makes the process more WAF and I think essential to ensure foolproof operation. Example: With such products it is possible to ask the electrician during construction to install a single switch per room (without two-way switch and without remote control switch) and to add additional wireless remote controls and thus reduce the cost of the installation bill.

In addition, the direct link Bindings works but the Aqara WXKG15LM remote control (WRS-R02) does not currently allow you to dissociate the right/left switch and that‚Äôs a shame because the triggering is instantaneous. **which makes the strong function interesting but **makes this device half functional.
