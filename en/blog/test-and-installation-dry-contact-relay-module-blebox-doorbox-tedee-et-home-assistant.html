    Following the article on the Tedee GO connected lock. The manufacturer Tedee also provided me with a relay module doorbox v2, it is a NO/NC module from the Blebox brand. Polish manufacturer of micro wifi module, I must admit that they are qualitative and there is rather interest because it is sold 69€ when the competition does the same thing for half the price!.

    Why did you provide me with this module, well you must suspect it is compatible with the Tedee environment and at the same time Home Assistant. Finally compatible with the Tedee environment yes but especially via the portal, because on the Tedee Smartphone application you can simply activate the NO/NC relay.

    This module is sold for around €69 to make NO/NC contact and limited to an input voltage ranging from 12 to 24v. It’s frankly expensive especially since competition like Tuya can sell it to you 2 to 3 times cheaper with wifi or Zigbee.

    For this price I hope that this Blebox module doorbox v2 is of quality

    Prerequisites

    To follow this tutorial you will need:

    • the wBox application by Blebox for Google playstore or Apple AppStore
    • The Tedee app Google playstore or Apple AppStore
    • Home Assistant d’installé et opérationnel.
    • An 12-24v power supply

    Like you, I find that there are a lot of applications to install, but if you use the Tedee or Home Assistant connected locks, the Blebox application will only be installed temporarily while you configure the wifi.

    Wifi settings

    Go to the wBox application by Blebox.eu and register. And yes for this tutorial you will need to install wBox in addition to Tedee on your Smartphone but this is temporary.

    Start by logging into the app so you can use it.

    Then connect to the module by selecting the wifi network named doorBox_v….. in the wifi settings of your smartphone.

    Once connected to the module, let’s configure it on the wifi of your home network. Go to Connection then click on the wifi network in Available WIFI Networks and enter the password of your network.

    Next step you will have to add this module in the application, return to the home page, select add new devices and Add device already connected to wifi Network. At this stage the Blebox module doorbox v2 is in the application.

    Last step, I will show you how to disable the initial wifi access point named doorBox_v…. Return to the module settings, then click on service connection (AP) and deactivate Access point.

    From now on you no longer need the wBox phone application by Blebox.eu

    ##WebUI

    Each Blebox module directly has a Web interface in order to configure the module’s settings, to access them you just need to retrieve the IP address of the module presented in the *wBox Smartphone app . You will have access *to all the data and more and you will be able to modify any parameters of the module. Awesome !

    Configuration in the Tedee Application

    Now let’s integrate the Blebox module doorbox v2 into the Tedee smartphone application. Nothing could be simpler, go to settings > integration > Blebox, authorize Tedee to have access to Blebox, and that’s it!

    Now on the main page of the application you will have the Blebox module(s) installed.

    You can control the opening or closing of the doorbox v2 relay from the application, not bad!

    Integration of Blebox modules in Home Assistant

    In HA it’s just as simple, click on the link below and in the devices you should see the Blebox module appear.

    All that remains is to click on configure and the doorbox v2 can be ordered via Home Assistant ‘‘Royal,,

    The interface is correctly recognized in Home Assistant if the commands do not match with Home Assistant does not hesitate to control the Blebox settings doorbox v2 by directly accessing the WebUI

    Conclusion

    The Blebox doorbox v2 is a complementary module to the world of connected locks and will allow you to not only control the front door, but also to control a garage door or gate. Integratable as desired into the Tedee application and Home Assistant, making this module complete. Negative point, the price, it is sold for €69, it is expensive for a micro module of this type, if you use Home Assistant you can easily move towards a Multifunction Nodon at €49 or other.
